1. IndieClass

React-Native / Express / PostgreSQL / Redux

IndieClass is a mobile app that allows independent instructors to set up classes, and allows users to find and sign up to these classes.

The project is built on React Native, and uses Google OAuth and Google Places APIs to handle authentication and map based searches. The backend is running on Express with a Postgres Database storing user data and class data.

This app was built by Paris Zhang and Sebastian Chamley and myself over 2 weeks.

To try it out, download the Expo app, follow the link above and scan the QR code using the Expo app.

2. Feedbackr

React / Express / MongoDB / SASS

An open source project whose purpose was to make the creation and sharing of assignments easier in this online age. Teachers shouldn't need to spend time building quizzes one by one when they can simply be shared between teachers.

The project was built on React, uses a NoSQL database hosted on MongoDB Atlas to store the quizzes and was styled using SASS.
